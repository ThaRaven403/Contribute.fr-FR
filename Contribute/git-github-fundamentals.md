- <span data-ttu-id="8c129-120">*Branche* : Unité de stockage qui contient les fichiers et dossiers qui composent le jeu de contenu d’un projet.</span><span class="sxs-lookup"><span data-stu-id="8c129-120">*Branch*: A unit of storage that contains the files and  folders that make up a project's content set.</span></span> <span data-ttu-id="8c129-121">Les branches servent à séparer les flux de travail (généralement appelés « versions »).</span><span class="sxs-lookup"><span data-stu-id="8c129-121">Branches separate streams of work (typically referred to as versions).</span></span> <span data-ttu-id="8c129-122">Les contributions sont toujours apportées et limitées à une branche spécifique.</span><span class="sxs-lookup"><span data-stu-id="8c129-122">Contributions are always made and scoped to a specific branch.</span></span> <span data-ttu-id="8c129-123">Tous les dépôts contiennent une branche par défaut (la branche « principale ») et une ou plusieurs branches destinées à être à nouveau fusionnées avec la branche principale.</span><span class="sxs-lookup"><span data-stu-id="8c129-123">All repositories contain a default branch (typically named "master") and one or more branches that are destined to be merged back into the master branch.</span></span> <span data-ttu-id="8c129-124">La branche principale sert de version actuelle et de « seule source de vérité » pour le projet.</span><span class="sxs-lookup"><span data-stu-id="8c129-124">The master branch serves as the current version and "single source of truth" for the project.</span></span> <span data-ttu-id="8c129-125">Il s’agit du parent à partir duquel toutes les autres branches dans le dépôt sont créées.</span><span class="sxs-lookup"><span data-stu-id="8c129-125">It's the parent from which all other branches in the repository are created.</span></span>
